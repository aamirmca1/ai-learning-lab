# 🤖 Langchain + HuggingFace Agent

a begginer-friendly conversational AI agent with **LangChain 1.x**, **LangGraph**, and the **HuggingFace Inference API**. Designed to be learned and enhanced step by step.


## ✨ Features

- 💬 **Interactive chat**
- 🧠 **Conversation memory**
- 🔌 **HuggingFace powered**


## ⚙️ Setup

### Install dependencies
```bash
pip install -r requirements.txt
```

### Add your HuggingFace API key
Open `.env` and replace the placeholde:
```
HUGGINGFACEHUB_API_TOKEN="hf_....."
```
